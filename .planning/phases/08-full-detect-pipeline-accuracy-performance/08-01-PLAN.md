```go
// OmegaIndex computes the Omega index (Collins & Dent 1988) between a detected
// overlapping community result and a ground-truth partition expressed as [][]NodeID.
// Returns a float64 in [0, 1]. Returns 0.0 if the graph has fewer than 2 nodes.
func OmegaIndex(result OverlappingCommunityResult, groundTruth [][]NodeID) float64
```

**Algorithm (Collins & Dent pair-counting):**

1. Collect all unique node IDs from both result.NodeCommunities keys and groundTruth members into a single sorted slice `allNodes`.
2. If len(allNodes) < 2, return 0.0.
3. Build `gtMembership map[NodeID][]int` from groundTruth: for each community index i, for each node in groundTruth[i], append i to gtMembership[node].
4. For every unordered pair (u, v) where u < v from allNodes:
   - Count `tResult` = number of communities in result where BOTH u and v appear (iterate result.Communities, check membership of both)
   - Count `tGT` = number of communities in groundTruth where BOTH u and v appear (iterate groundTruth, check membership of both)
   - Increment `agree` if tResult == tGT
   - Track frequency: `freqResult[tResult]++`, `freqGT[tGT]++`
5. totalPairs = len(allNodes) * (len(allNodes) - 1) / 2
6. observed = float64(agree) / float64(totalPairs)
7. expected = sum over all k of (float64(freqResult[k]) * float64(freqGT[k])) / float64(totalPairs * totalPairs)
8. If expected >= 1.0, return 1.0 (degenerate case).
9. omega = (observed - expected) / (1.0 - expected)
10. Clamp to [0.0, 1.0] — negative values become 0.0.

**Optimization for step 4:** Instead of iterating all communities for each pair, precompute `resultMembership map[NodeID]map[int]struct{}` (node -> set of community indices) from result.Communities. Then for pair (u,v), tResult = count of community indices present in BOTH resultMembership[u] and resultMembership[v]. Same for gtMembership. This is O(C) per pair where C = max communities per node, not O(K) where K = total communities.

**Exact implementation of Detect():**

```go
func (d *egoSplittingDetector) Detect(g *Graph) (OverlappingCommunityResult, error) {
    // Guard: directed graphs not supported (per D-01 from CONTEXT.md)
    if g.IsDirected() {
        return OverlappingCommunityResult{}, ErrDirectedNotSupported
    }

    // Step 1: Build persona graph (Algorithms 1 + 2)
    personaGraph, _, inverseMap, err := buildPersonaGraph(g, d.opts.LocalDetector)
    if err != nil {
        return OverlappingCommunityResult{}, err
    }

    // Step 2: Run global detector on persona graph
    globalResult, err := d.opts.GlobalDetector.Detect(personaGraph)
    if err != nil {
        return OverlappingCommunityResult{}, err
    }

    // Step 3: Map persona communities back to original nodes (Algorithm 3)
    nodeCommunities := mapPersonasToOriginal(globalResult.Partition, inverseMap)

    // Step 4: Deduplicate community IDs per node
    for node, comms := range nodeCommunities {
        seen := make(map[int]struct{}, len(comms))
        unique := comms[:0]
        for _, c := range comms {
            if _, ok := seen[c]; !ok {
                seen[c] = struct{}{}
                unique = append(unique, c)
            }
        }
        nodeCommunities[node] = unique
    }

    // Step 5: Build Communities [][]NodeID from NodeCommunities
    // First pass: find max community ID to size the slice
    maxComm := -1
    for _, comms := range nodeCommunities {
        for _, c := range comms {
            if c > maxComm {
                maxComm = c
            }
        }
    }

    communities := make([][]NodeID, maxComm+1)
    for node, comms := range nodeCommunities {
        for _, c := range comms {
            communities[c] = append(communities[c], node)
        }
    }

    // Remove empty community slots (sparse community IDs from global detector)
    var filtered [][]NodeID
    commRemap := make(map[int]int) // old index -> new index
    for i, members := range communities {
        if len(members) > 0 {
            commRemap[i] = len(filtered)
            filtered = append(filtered, members)
        }
    }

    // Remap NodeCommunities indices to match filtered Communities
    for node, comms := range nodeCommunities {
        remapped := make([]int, len(comms))
        for j, c := range comms {
            remapped[j] = commRemap[c]
        }
        nodeCommunities[node] = remapped
    }

    return OverlappingCommunityResult{
        Communities:     filtered,
        NodeCommunities: nodeCommunities,
    }, nil
}
```

**Critical details:**
- Deduplication in Step 4 is necessary because `mapPersonasToOriginal` may produce duplicate community IDs when multiple personas of the same node land in the same global community.
- Step 5 compacts sparse community IDs into contiguous indices so `Communities[i]` has no nil gaps.
- The `commRemap` ensures `NodeCommunities` indices are consistent with the filtered `Communities` slice.
