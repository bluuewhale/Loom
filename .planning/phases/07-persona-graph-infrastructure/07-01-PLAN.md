    ```go
    // buildEgoNet returns the ego-net of node v: the subgraph induced by v's
    // neighbors, excluding v itself. This implements Algorithm 1 of the Ego
    // Splitting framework.
    func buildEgoNet(g *Graph, v NodeID) *Graph {
        neighbors := g.Neighbors(v)
        nodeIDs := make([]NodeID, 0, len(neighbors))
        for _, e := range neighbors {
            nodeIDs = append(nodeIDs, e.To)
        }
        return g.Subgraph(nodeIDs)
    }
    ```
    Key: pass only neighbor To-fields to Subgraph. v is excluded because it is never in nodeIDs. Per EGO-CRIT-01.

    **buildPersonaGraph signature:**
    ```go
    // buildPersonaGraph constructs the persona graph from g using the given local
    // detector. For each node v, it builds the ego-net, detects local communities,
    // and creates one persona node per (v, community) pair. Edges are rewired
    // bidirectionally per Algorithm 2 Section 2.2.
    //
    // Returns:
    //   - personaGraph: new undirected graph with persona nodes and rewired edges
    //   - personaOf: map[NodeID]map[int]NodeID -- original node -> community -> PersonaID
    //   - inverseMap: map[NodeID]NodeID -- PersonaID -> original NodeID
    func buildPersonaGraph(g *Graph, localDetector CommunityDetector) (*Graph, map[NodeID]map[int]NodeID, map[NodeID]NodeID, error)
    ```

    Implementation steps inside buildPersonaGraph:
    1. Scan g.Nodes() to find maxNodeID. Set nextPersona = maxNodeID + 1.
    2. Initialize personaOf = map[NodeID]map[int]NodeID{}, inverseMap = map[NodeID]NodeID{}.
    3. For each node v in g.Nodes():
       a. egoNet := buildEgoNet(g, v)
       b. If egoNet.NodeCount() == 0, create single persona for v with community 0: personaOf[v] = map[int]NodeID{0: nextPersona}, inverseMap[nextPersona] = v, nextPersona++. Continue.
       c. result, err := localDetector.Detect(egoNet). Handle error.
       d. Collect unique community IDs from result.Partition values.
       e. For each unique community c: personaOf[v][c] = nextPersona, inverseMap[nextPersona] = v, nextPersona++.
    4. Build personaGraph = NewGraph(false).
    5. Add all persona nodes: for each entry in inverseMap, personaGraph.AddNode(personaID, 1.0).
    6. Wire edges with dedup. For each node u in g.Nodes(), for each edge (u, v) in g.Neighbors(u):
       a. Use seen map[[2]NodeID]struct{} with canonical (lo,hi) = (min(u,v), max(u,v)) to skip if already processed.
       b. Look up c_v = partition_u[v] (v's community in G_u's partition). If v absent from partition_u, skip.
       c. Look up c_u = partition_v[u] (u's community in G_v's partition). If u absent from partition_v, skip.
       d. persona_u = personaOf[u][c_u], persona_v = personaOf[v][c_v].
       e. personaGraph.AddEdge(persona_u, persona_v, edgeWeight).
    7. Return personaGraph, personaOf, inverseMap, nil.

    IMPORTANT: Store each node's partition in a map during step 3: partitions map[NodeID]map[NodeID]int (node v -> its ego-net partition). This is needed in step 6 for cross-lookups.
