
**graph/louvain.go — buildSupergraph canonical dedup (lines 304-339):**

The current code iterates ALL adjacency entries (both directions of each undirected edge) and divides accumulated weights by 2. Replace with single-pass dedup using endpoint ordering.

Replace the accumulation loop (lines 310-325) with:
```go
for _, n := range g.Nodes() {
    superN := commToSuper[partition[n]]
    for _, e := range g.Neighbors(n) {
        superNeighbor := commToSuper[partition[e.To]]
        if superN == superNeighbor {
            // Intra-community edge -> self-loop on supernode.
            // For undirected graphs, each edge (u,v) appears in both adjacency[u] and adjacency[v].
            // Process only when n <= e.To to count each edge once.
            // n == e.To handles actual graph self-loops.
            if n <= e.To {
                selfLoops[superN] += e.Weight
            }
        } else {
            // Inter-community edge. Process only from the lower-community endpoint.
            // Use original node IDs for dedup (not super IDs) since canonical key already handles direction.
            if n < e.To {
                a, b := superN, superNeighbor
                if a > b {
                    a, b = b, a
                }
                interEdges[edgeKey{a, b}] += e.Weight
            }
        }
    }
}
```

CRITICAL: For self-loops, use `n <= e.To` (not `n < e.To`) because actual graph self-loops have `n == e.To` and must be counted.
For inter-community edges, use `n < e.To` (strict) since there are no self-edges between different communities.

Then update the write section (lines 328-339) to remove the `/2.0` division:
```go
// Self-loops: each edge counted once (lo<=hi guard).
for super, w := range selfLoops {
    newGraph.AddEdge(super, super, w)
}

// Inter-community edges: each edge counted once (lo<hi guard).
for key, w := range interEdges {
    newGraph.AddEdge(key.a, key.b, w)
}
```

Pre-size the maps for fewer rehashes:
```go
interEdges := make(map[edgeKey]float64, g.EdgeCount()/2)
selfLoops := make(map[NodeID]float64, len(commList))
```

Update the comments at lines 330-331 and 335-337 to reflect the new single-pass approach (remove references to "counted twice" and "divide by 2").
