<objective>
Implement the three highest-impact optimizations and dead code cleanup for the graph core.

1. **Nodes() cache** — Cache `sortedNodes []NodeID` on Graph struct, invalidate on mutation, copy-before-shuffle in phase1.
2. **rand.Rand reuse via math/rand/v2** — Migrate louvainState and leidenState from math/rand to math/rand/v2 with stored PCG source for zero-alloc reseed.
3. **Dead code removal** — Delete deltaQ, newLouvainState, newLeidenState; add O(n) comment to CommStrength; add comment to Tolerance field noting it is not yet implemented.

Purpose: Nodes() caching eliminates the dominant allocation source (~15-25 x 10K-element slices per Detect). rand/v2 PCG reuse eliminates 2-3 allocs per state reset. Dead code removal reduces maintenance surface.

Output: Modified graph.go, louvain.go, louvain_state.go, leiden_state.go, detector.go
</objective>

func (g *Graph) Nodes() []NodeID {
    if g.sortedNodes != nil {
        return g.sortedNodes
    }
    ids := make([]NodeID, 0, len(g.nodes))
    for id := range g.nodes {
        ids = append(ids, id)
    }
    slices.Sort(ids)
    g.sortedNodes = ids
    return ids
}
