```go
package graph

import (
    "math/rand"
    "time"
)

// louvainState holds mutable state for a single Louvain detection run.
type louvainState struct {
    partition map[NodeID]int    // node -> community ID
    commStr   map[int]float64  // community ID -> sum of node strengths (cached)
    rng       *rand.Rand       // per-run RNG for node shuffle
}

// newLouvainState initializes state with each node in its own community.
// Community IDs start at 0 and are assigned in Nodes() order.
func newLouvainState(g *Graph, seed int64) *louvainState {
    var src rand.Source
    if seed != 0 {
        src = rand.NewSource(seed)
    } else {
        src = rand.NewSource(time.Now().UnixNano())
    }

    nodes := g.Nodes()
    partition := make(map[NodeID]int, len(nodes))
    commStr := make(map[int]float64, len(nodes))

    for i, n := range nodes {
        partition[n] = i
        commStr[i] = g.Strength(n)
    }

    return &louvainState{
        partition: partition,
        commStr:   commStr,
        rng:       rand.New(src),
    }
}
```

Key design points:
- `commStr` cache avoids O(n) `CommStrength` calls in the hot loop (per RESEARCH Pitfall 2)
- `rng` is per-run, seeded from opts.Seed (Seed=0 -> time-based random)
- Each node starts in its own community (singleton partition), community ID = index in Nodes() slice

**1. `func (d *louvainDetector) Detect(g *Graph) (CommunityResult, error)` — main entry point**

Guard clauses at top (per LOUV-05 and CONTEXT.md error contract):
- `g.IsDirected()` -> return `CommunityResult{}, ErrDirectedNotSupported`
- `g.NodeCount() == 0` -> return `CommunityResult{}, nil`
- `g.NodeCount() == 1` -> return `CommunityResult{Partition: map[NodeID]int{g.Nodes()[0]: 0}, Modularity: 0.0, Passes: 1, Moves: 0}, nil`
- `g.TotalWeight() == 0` -> all nodes disconnected: return singleton partition (each node its own community), Q=0.0, Passes=1, Moves=0

Resolve zero-value options per CONTEXT.md locked decisions:
- `Resolution == 0.0` -> 1.0
- `Tolerance == 0.0` -> 1e-7
- `MaxPasses == 0` -> unlimited
- `Seed == 0` -> random

Outer convergence loop (LOUV-03):
- Initialize `louvainState` via `newLouvainState(currentGraph, seed)`
- Call `phase1` on currentGraph; collect moves
- If moves == 0 -> break (converged)
- If MaxPasses > 0 and totalPasses >= MaxPasses -> break
- Call `buildSupergraph` to compress communities into supernodes (LOUV-02)
- Update `nodeMapping` (maps original NodeIDs through supergraph chain) — per RESEARCH Pitfall 4
- Re-initialize state on the new supergraph and repeat

After loop:
- Reconstruct final partition by composing nodeMapping chain back to original NodeIDs
- Call `normalizePartition` to make community IDs 0-indexed contiguous
- Compute final Q via `ComputeModularityWeighted(g, finalPartition, resolution)`
- Return CommunityResult with Partition, Modularity, Passes, Moves

**2. `func phase1(g *Graph, state *louvainState, resolution, m float64) int` — local moves (LOUV-01)**

One full pass over all nodes in shuffled order (using state.rng.Shuffle):
- For each node `n`:
  - Get currentComm = state.partition[n]
  - Remove n from current community: state.commStr[currentComm] -= g.Strength(n)
  - Scan neighbors to find candidate communities (deduplicate via map[int]struct{})
  - Also include currentComm as candidate (node might stay)
  - For each candidate comm: compute gain = deltaQ(n, comm) - deltaQ(n, currentComm)
  - Pick bestComm with highest gain > 0; if no gain > 0, stay in currentComm
  - Assign state.partition[n] = bestComm; state.commStr[bestComm] += g.Strength(n)
  - If bestComm != currentComm: moves++
- Return moves count

**3. `func deltaQ(g *Graph, n NodeID, comm int, partition map[NodeID]int, commStr map[int]float64, resolution, m float64) float64` — modularity gain (LOUV-04)**

Exact formula matching ComputeModularityWeighted convention:
```go
func deltaQ(g *Graph, n NodeID, comm int, partition map[NodeID]int,
    commStr map[int]float64, resolution, m float64) float64 {
    kiIn := g.WeightToComm(n, comm, partition)
    sigTot := commStr[comm]
    ki := g.Strength(n)
    twoM := 2.0 * m
    return kiIn/m - resolution*(sigTot/twoM)*(ki/twoM)
}
```

The `m` here is `g.TotalWeight()`. The denominator convention matches `ComputeModularityWeighted` which uses `twoW = 2 * g.TotalWeight()` for the degree-product term.

**4. `func buildSupergraph(g *Graph, partition map[NodeID]int) (*Graph, map[NodeID]NodeID)` — phase 2 compression (LOUV-02)**

- Assign contiguous supernode IDs to each distinct community
- Accumulate inter-community edge weights using canonical `[2]NodeID{min, max}` key (RESEARCH anti-pattern: avoid double-count)
- Accumulate intra-community (self-loop) weights separately
- Divide all accumulated weights by 2 (undirected adjacency lists each edge in both directions)
- Build new graph via `NewGraph(false)` + `AddEdge`
- Return supergraph and mapping: `superNodeID -> []originalNodeID` (or simpler: map each original node to its supernode)
- The second return value maps supernode IDs back to a representative original NodeID for partition reconstruction

**5. `func normalizePartition(partition map[NodeID]int) map[NodeID]int` — 0-indexed contiguous (CONTEXT.md locked decision)**

Remap community IDs to {0, 1, 2, ...} in encounter order.
